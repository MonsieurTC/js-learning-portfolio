# JS Learning Portfolio

## Objectif
Ce dépôt documente mon apprentissage progressif de JavaScript.
Chaque concept étudié est illustré par du code, des notes et des exemples.

## Méthode
- Discipline : 45 minutes par jour (méthode B)
- Apprentissage actif : comprendre → coder → expliquer
- Parking stratégique pour idées et questions

## Organisation du dépôt
- `index.html` : page principale avec sections par notion
- `style.css` : styles simples pour visualiser les sections
- `script.js` : code JavaScript pour chaque concept
- Chaque jour : commit avec ajout d’une nouvelle section et exemples
