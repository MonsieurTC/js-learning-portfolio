Variables & Types - Jour 1

Objectif : comprends let, const, var et les types primitifs (string, number, boolean, null, undefined).

Explications

Les variables sont des conteneurs pour stocker des données. En JavaScript, il existe trois façons de déclarer une variable : let, const et var.

let : variable pouvant changer de valeur

const : variable dont la valeur ne peut pas changer

var : ancienne façon de déclarer une variable, variable dépréciée, moins utilisée dans les codes modernes. Ne pas être surpris de le trouver dans de vieux codes.

Exemples pratiques

// Exemple simple de code
let nom = "Richard";
const age = 46;
var pays = "France";

console.log(nom, age, pays);
